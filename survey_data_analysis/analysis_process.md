# 价值观聚类分析过程记录

## 目标

对中国人价值观调研数据（5852人，425个字段）做聚类分析，提炼典型价值观人群及其比例，作为后续构建虚拟人的基础。

---

## 数据预处理

### 字段筛选

从 39 个 group 中，选取以下 20 个价值观/态度相关 group 作为聚类输入：

| 类别 | Groups |
|---|---|
| 自我认知 | SEQ01, SEQ02, SEQ03, SEQ05 |
| 他人与社会 | OTQ01, OTQ02, OTQ03, OTQ04, OTQ05, OTQ06, OTQ07, OTQ08, OTQ10 |
| 国家与集体 | GRQ01, GRQ02, GRQ07, GRQ08, GRQ09 |
| 财富观 | MAQ01, MAQ02 |

排除标准：
- 多选题（无法直接数值化）
- 开放/数值填写题（年龄、城市名等）
- 名义分类题（性别、有无小孩等）
- 888/999/-1 等 sentinel 值 → 视为缺失
- 超出量表范围的值 → 视为缺失

最终保留 **166 个 Likert 字段**，缺失率 >40% 的字段和 >50% 缺失的受访者被删除，剩余 **5850 人**。

---

## 方案一：直接 GMM（失败）

### 做法
- 对 166 个标准化特征直接做 GMM（covariance_type="diag"）
- 测试 K=3..8，按 BIC 选最优 K

### 结果（K=8 最优）

| Class | 比例 | 初步标签 |
|---|---|---|
| 0 | 4.7% | 国际乐观 + 传统家庭观 |
| 1 | 9.7% | 低国家认同、社会不公平感强 |
| 2 | 0.7% | **异常类**（中医相关字段均值13+，量表最大5） |
| 3 | 18.5% | 进步价值观：性别平等、LGBT友好 |
| 4 | 13.0% | 爱国传统型 |
| 5 | 3.5% | **异常类**（GRQ0807均值46，量表最大5） |
| 6 | 19.8% | 传统婚育观、婚姻保守 |
| 7 | 30.3% | 主流爱国型 |

### 问题
- Class 2 和 Class 5 是**数据异常值**导致的噪音类，修复后消失
- 修复后重跑 K=7，财富观/自我评价维度信号极弱，被国家认同和婚育观维度压制

---

## 方案二：PCA 降维 + GMM（效果不理想）

### 做法
- 先用 PCA 把 166 个特征压缩到 69 个组件（解释 70% 方差），再做 GMM

### PCA 观察

| PC | 方差% | 语义 |
|---|---|---|
| PC1 | 9.3% | 国家认同（中国形象、传统文化自信） |
| PC2 | 8.1% | 性别保守主义（男主外女主内、男性优越） |
| PC3 | 2.8% | LGBT 态度 + 婚育自由 |
| PC4 | 2.5% | 社会安全感 + 生活满足感 |
| PC5 | 2.3% | 国际关系乐观程度 |
| PC6 | 2.0% | 时间取向（眼前 vs 长远） |
| PC7 | 1.5% | 自我效能感 vs 迷茫感 |
| PC8 | 1.4% | 冒险/刺激寻求倾向 |
| PC9 | 1.4% | 国防/军事重视度 |
| PC10 | 1.3% | 当下享乐 vs 长期规划 |
| PC11-15 | <1%/个 | 权威服从、消费观、比较心理等 |

**关键发现**：PC1+PC2 解释 17%，之后每个 PC 不足 3%，呈长尾分布。财富观、自我评价分散在多个 PC 中，没有形成集中的单一因子。

### GMM 结果（K=3 最优，BIC）

BIC 在 69 维空间下随 K 线性增长，强制选出 K=3，过于粗粒度：

| Class | 比例 | 标签 |
|---|---|---|
| 0 | 25.2% | 功利/实用型：财富观强、性别保守、命定论 |
| 1 | 28.3% | 爱国传统型：国家自豪、传统文化、集体主义 |
| 2 | 46.5% | 现代温和型：各维度偏低、LGBT 相对友好 |

### 问题
- 这份数据**没有少数几个强势正交因子**，PCA 压缩后丢失了大量有效信息
- BIC 在高维空间惩罚过重，把 K 压到最小值
- K=3 太粗，对构建有区分度的虚拟人来说不够用

---

## 方案三：Group 归一化 + K-means（进行中）

### 动机
- 直接用原始 166 个特征，但按 group 做归一化，使财富观、自我评价等 group 与国家认同权重平等
- 用 K-means 代替 GMM，在高维空间更稳定
- 用 silhouette score 和 elbow 法选 K，不依赖 BIC

### 预期改善
- 财富观、自我评价等维度不再被强势维度压制
- 可以测试更多 K 值，找到语义更丰富的分类

---

## 核心结论（截至目前）

1. **数据结构**：价值观信号分散，没有几个主导因子，这反映了中国人价值观的多元性
2. **维度压制**：国家认同（GRQ07）和性别/婚育观（OTQ05/06/07）信号最强，容易压制财富观、自我认知等维度
3. **Group 归一化**是解决维度不均衡的关键步骤
4. **异常值处理**：量表范围过滤比 sentinel 过滤更重要，必须两者都做
